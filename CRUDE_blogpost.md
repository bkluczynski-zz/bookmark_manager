The CRUD stands for CREATE, READ, UPDATE and DELETE

The SQL stands for Structured Query Language

To create a database, type : CREATE DATABASE demodb1 WITH OWNER demorole1 ENCODING 'UTF8';

To read a database, type SELECT * FROM <tablename>

To update a table in a database, type UPDATE table_name
SET column_name_1 = new_value_1, column_name_2 = new_value_2
WHERE column_name = some_value

To delete from a database, type : DELETE FROM table_name
WHERE column_name = some_value
